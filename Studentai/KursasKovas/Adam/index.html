HTML – tai paprasta ir lengvai išmokstama kalba tinklapiams kurti. Naujausia šios kalbos versija pastaruoju metu tampa viena populiariausių kalbų multimedijai perduoti. Su keliolika šios kalbos žymų galima sukurti visą tinklapį. Dauguma šiuolaikinių HTML rašyklių yra pakankamai funkcionalios, todėl nebereikia rašyti ilgų puslapių kodų.

CSS programavimo kalba išplečia HTML kalbos galimybes. Su šia kalba galima atvaizduoti teksto šriftą, tinklapio išdėstymą, spalvas, atstumą tarp elementų. CSS kalba leidžia tą patį stiliaus šabloną naudoti visuose tinklapio puslapiuose.

MYSQL – tai standartizuota duomenų bazių valdymo kalba. Ji yra skirta redaguoti ir rasti informaciją, esančią tam tikrose duomenų bazių valdymo sistemose. Šiuolaikinei internetinei svetainei sukurti vien HTML kalbos neužteks. Būtent „Javascript“ kalba padės sukurti patogią naudoti, dinamišką svetainę. Su šia kalba galima kurti išskleidžiamuosius meniu, patvirtinimo lenteles, animaciją ir pan. – viską, kas suteiktų pridėtinės vertės jūsų svetainei.
